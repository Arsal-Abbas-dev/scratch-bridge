# Scratch Gap Maze

Scratch Gap Maze is a web-based programming education tool designed to help beginners transition gradually from block-based programming to Python-style text-based code.

The project focuses on the gap many beginners experience after using visual tools such as Scratch and then moving to text-based languages such as Python. Instead of forcing learners to abandon blocks immediately, this platform will use maze-based challenges and a hybrid block/text workspace where selected blocks can eventually be replaced with equivalent Python-style code.

## Project Goal

The goal is to build a focused prototype that helps beginners:

- learn sequencing, loops, conditionals, functions, and debugging
- see how block-based logic translates into Python-style code
- gradually replace visual blocks with text
- receive beginner-friendly explanations when errors occur

## Planned Technical Features

- React and TypeScript frontend
- Google Blockly workspace
- Maze-based programming challenges
- Custom maze execution engine
- Internal program representation
- Live Python-style code view
- Hybrid block/text transition system
- Beginner-friendly error messages
- User testing with beginner programmers

## Version 1 Scope

Version 1 will focus on a small but complete prototype. It will not attempt to become a full coding platform.

Included in Version 1:

- sequencing
- loops
- conditionals
- simple functions
- debugging support
- block-to-code translation
- hybrid block/text transition
